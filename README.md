# Customer Shopping Behavior Analysis (Power BI)
data analytics & dashboard project analyzing customer shopping behavior.
The goal is to explore patterns in customer purchases, demographics, purchase frequency, and spending trends, and present insights using Python for analysis and Power BI for visualization.

# About the Data
The dataset is used to analyze customer behavior trends, purchasing frequency, and spending patterns,
and to support data visualization and dashboard creation in Power BI.

| Column Name                | Description                                           |
|----------------------------|-------------------------------------------------------|
| customer_id                | Unique identifier for each customer                  |
| age                        | Customer age                                         |
| gender                     | Customer gender                                      |
| item_purchased             | Item purchased by the customer                       |
| category                   | Product category                                     |
| purchase_amount            | Purchase amount in USD                               |
| frequency_of_purchases     | Purchase frequency (Weekly, Monthly, etc.)           |
| review_rating              | Customer review rating                               |
| discount_applied           | Indicates whether a discount was applied             |
| promo_code_used            | Indicates whether a promo code was used              |
